## 安装
> 提示  
> 请确保在开始之前已经安装了 [Node.js](https://www.nodejs.org) 和 [Git](https://git-scm.com/)，此外我们建议使用 Linux 而不是 Windows 进行配置。

``` bash
# 安装依赖
$ yarn

# 启动服务器，支持热重载
$ yarn dev

# 为生产环境编译并启动服务器
$ yarn build
$ yarn start
```
